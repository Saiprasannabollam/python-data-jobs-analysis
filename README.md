# Python Data Jobs Analysis

## Overview

This repository contains a comprehensive Python-based data analysis project exploring the global data jobs market. Using real-world job posting data, the project analyzes salary trends, in-demand skills, optimal skills for career growth, and employment patterns through exploratory data analysis and visualizations.

---

## Dataset

The dataset used in this project is publicly available and exceeds GitHub's web upload size limit. It has not been included in this repository. The analysis notebooks can be used with the original dataset.

## Project Objectives

- Explore the structure of the dataset.
- Analyze salary distributions across data-related roles.
- Identify the most in-demand technical skills.
- Discover high-paying and high-demand skills.
- Visualize trends to support data-driven career insights.
